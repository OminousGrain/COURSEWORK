The top of each page features a JavaScript powered carousel, this is a visual feature that improves the professional quality of the site. The carousel displays tech related images at intervals.


Below the carousel is a navigation bar with drop down menus that highlight upon hovering. This is a feature that demonstrates understanding of CSS animation features. Similarly, all of the links in the navigation bar transport the user to the relevant pages.


The main content of the site could be styled more, with paragraphs being laid out in a more visually appealing manner. However, the imported font and ‘cream-dust’ background image help give the site a sense of being of a higher quality.


The titles in the webpage highlight by growing when hovered over, and the paragraphs similarly change their background color. This is a feature to demonstrate knowledge of interactivity, however, I think it is a bit garish and shouldn’t be used in a real product.


At the bottom of the page is a footer that contains a copyright notice. This footer scrolls with the user, remaining fixed in place at the bottom of the screen.This is both to highlight awareness of CSS and of copyright standards in designing webpages.
