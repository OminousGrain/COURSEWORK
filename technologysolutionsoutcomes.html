Using Trello

Trello is an invaluable tool for planning large projects. In producing this coursework I utilised Trello extensively. It was utilised for monitoring the projects scope, for orienting tasks and for formatting basic versions of the site. Beginning with basic framework of the site, providing pages with the relevant content, up to providing a more stylised and presentable finished site.

Using GitHub

Github was used to provide access to the site files from multiple locations, allowing me to work on the project from multiple locations. Similarly, it allowed me to keep the project updated as new versions of the site were tested and then pushed to the master branch.
