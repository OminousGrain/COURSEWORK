Visual Studio Code

VS Code was perfectly suitable for the task of designing and debugging html, CSS, and javascript. Its tools were able to significantly reduce the time it took to actualise a functioning site.

Chrome Browser

The Chrome browser was perfectly suitable for testing out various prototypes of the site as it has well maintained features that enabled me to try out ideas without fear of them not being supported

Trello

Trello was invaluable for organising my tasks and objectives.Its framework was clear and concise and allowed me a clear overview of what my goals and objectives were at any given time.

GitHub

GitHub enabled me to regularly update my site and access it from different location. In terms of 'ease of access' it was extremely useful and fit for purpose.
